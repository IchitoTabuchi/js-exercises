# 手順

"https://developer.microsoft.com/en-us/graph/graph-explorer"にアクセスし、以下の手順でアクセスキーを取得する。

1. Microsoftアカウントでサインインする。
2. "Access token" タブを押下する。
3. トークンをコピーする。

![alt text](image.png)

4. コピーしたトークンを Web アプリのフォームに貼り付ける。

![alt text](image-1.png)

# 実行結果

画像ファイルを選択してアップロードを行った。
