# このようないわゆるメッセージパッシングによって排他制御処理相当を行う並行処理モデルを何と呼ぶか書きなさい。

## 名称

アクターモデル (Actor Model)

### 特徴

- 各スレッド(アクター)は独立した状態を持ち、他のスレッドから直接アクセスされない。
- スレッド間で共有メモリを使わず、メッセージの送受信によってデータの更新を行う。
- 排他制御(ロック)が不要で、デッドロックやレースコンディションのリスクが低い。
